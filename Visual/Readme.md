# Концепия экспертной системы

1. Входные данные:
    1. Тип шкалы 
        - Номинальные: визуализация категорий
        - Интервальная - визуализация количественных велечин
    2. Фактор времени:
        - Временные ряды
        - Одномоментные данные
    3. Векторизация:
        - Один параметр
        - Несколько параметров

2. Выбор алгоритма по цели\                                                             Тип шкалы
    - Сравнение и ранжировка: Ленейчатая или гистограмма.                               номинальная шкала
    - Изменение по времени: линейный график.                                            шкала отношений + динамика
    - Часть к целому: Pie Chart\ Treemap.                                               1 параметр + сумма 100%
    - Корреляция: Scatter plot(2переменных)\ bubble chart(3переменных).                 2 и более числовых параметра
    - Распределение: Higstogram\Box plot.                                               1 числовой параметр
    - Отклонение: двустороняя линейчатая.                                               Сравнение с нулем
    - География: Картограмма.                                                           координаты

3. Алгоритм принятия решения
    - Проверка на иерархию:
        - есть ли в данных вложенность (родитель-потомок): да - используем Treemap\Sunburst
    - Проверка на время:
        - является ли время ключевой осью: да; много точек? да Line, нет Column
        Важна ли дилтельность процессов? да: Gantt Chart
    - Проверка на кол-во переменных: 
        - Одна - гистограмма или сравнение
        - 2 - scatter plot\ линейный график
        - более 2х - radar chart
